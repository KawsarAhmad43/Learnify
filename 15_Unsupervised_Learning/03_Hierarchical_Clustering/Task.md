# Task: Animal Clustering

## Objective
Use Hierarchical Clustering to group animals based on features.

## Data
Create a small dataset:
*   Features: `Legs`, `Teeth`, `Tail` (1=Yes, 0=No).
*   Animals: Dog, Cat, Spider, Human, Snake, Kangaroo.

## Tasks
1.  **Define Data**: Create a DataFrame or Array.
2.  **Dendrogram**: Plot the dendrogram.
3.  **Insight**: Does the dendrogram group Dog and Cat together? Does it separate the Spider?
