# Task: Salary Prediction

## Objective
Build a simple linear regression model.

## Data
Create a dataset:
*   **Experience (Years)**: [1.1, 1.3, 1.5, 2.0, 2.2, 2.9, 3.0, 3.2, 3.2, 3.7]
*   **Salary**: [39343, 46205, 37731, 43525, 39891, 56642, 60150, 54445, 64445, 57189]

## Tasks
1.  **Plot**: Scatter plot of Experience vs Salary.
2.  **Train**: Fit a Linear Regression model.
3.  **Predict**: What is the predicted salary for someone with **5.5 years** of experience?
