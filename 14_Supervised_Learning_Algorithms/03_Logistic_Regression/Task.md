# Task: Purchase Prediction

## Objective
Predict if a customer will buy a product.

## Data
*   **Age**: [22, 25, 47, 52, 46, 56, 55, 60, 62, 63]
*   **Bought**: [0, 0, 1, 0, 1, 1, 1, 1, 1, 1]

## Tasks
1.  **Train**: Fit a Logistic Regression model (Age -> Bought).
2.  **Probability**: What is the probability that a **40-year-old** will buy?
3.  **Prediction**: Will a **30-year-old** buy? (Yes/No).
