# Task: Digits Classification Showdown

## Objective
Beat the single decision tree using an ensemble.

## Data
`sklearn.datasets.load_digits()`

## Tasks
1.  **Baseline**: Train a single `DecisionTreeClassifier`. Record accuracy.
2.  **Ensemble**: Train a `RandomForestClassifier` with 50 trees. Record accuracy.
3.  **Compare**: How much did the ensemble improve performance?
