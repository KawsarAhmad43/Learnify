# Task: Derivatives & Gradients

## Objective
Practice calculating partial derivatives and gradients using Python.

## Tasks

### Task 1: Partial Derivatives
Given the function:
$$f(x, y) = 3x^2 + 2y^3 - 5xy$$

1.  Use `sympy` to calculate $\frac{\partial f}{\partial x}$ and $\frac{\partial f}{\partial y}$.
2.  Evaluate these partial derivatives at the point $(x=1, y=2)$.

### Task 2: Gradient Vector
1.  Define the function $g(x, y) = \sin(x) \cdot \cos(y)$.
2.  Compute the gradient vector $\nabla g$.
3.  Visualize the function using a 3D plot or a contour plot.
