# Task: Housing Data Visualization

## Objective
Explore a housing dataset using Seaborn.

## Data
Create a DataFrame:
*   **Price**: Random normal (Mean=300k, Std=100k).
*   **Area**: Random normal (Mean=2000, Std=500).
*   **Type**: ['Apartment', 'House', 'Villa'].
*   **Has_Pool**: [0, 1].

## Tasks
1.  **Distribution**: Plot the distribution (Hist+KDE) of **Price**.
2.  **Relationship**: Plot a Scatter plot of **Area vs Price**. Use 'Has_Pool' as the color (hue).
3.  **Comparison**: Plot a Boxplot of **Price** grouped by **Type**. Which property type is the most expensive on average?
