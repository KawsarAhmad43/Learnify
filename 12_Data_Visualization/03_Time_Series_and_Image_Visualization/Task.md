# Task: Time Series & Images

## Objective
Visualize trends and pixel data.

## Tasks

### Task 1: Temperature Trend
create a DataFrame with:
*   **Month**: Jan to Dec.
*   **Temp**: [30, 32, 45, 55, 65, 75, 80, 78, 70, 60, 50, 40]
1.  Plot a **Line Chart** of Temp vs Month with markers ('o').
2.  Add gridlines and labels.

### Task 2: Noise Image
1.  Generate a random noise image of shape `(100, 100)`.
2.  Display it using a 'hot' colormap (`cmap='hot'`).
3.  Add a Colorbar.
