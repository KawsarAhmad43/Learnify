# Task: Advanced Manipulation

## Objective
Master Grouping and Merging.

## Data
Create two DataFrames:
1.  `employees`: EmployeeID, Name, Department.
2.  `salaries`: EmployeeID, Year, Salary.

## Tasks

### Task 1: Department Stats
1.  Merge the tables to get a single DataFrame with [Name, Department, Year, Salary].
2.  Group by **Department** to find the **Average Salary** per department.
3.  Find the highest paid employee in each department.

### Task 2: Pivot
1.  Create a Pivot Table showing the **sum** of salaries, where rows are **Department** and columns are **Year**.
